RippleVMS Feature Overview 2 / 14

Role-Based Access Control

Four distinct user roles with appropriate permissions:

Volunteer

  • Browse and sign up for shifts
  • View personal dashboard
  • Access zone maps and POIs
  • Submit community sighting reports

Coordinator

  • Create and manage shifts
  • Confirm volunteer RSVPs
  • Send email blasts
  • Manage zone mapping

Dispatcher

  • Handle incoming sighting reports
  • Coordinate field responses
  • View all zone activity
  • Time-block based assignments

Administrator

  • Full system configuration
  • User management
  • Zone and shift type setup
  • Qualified roles management
RippleVMS Feature Overview 3 / 14

Volunteer Experience

Intuitive tools for community members

Personalized Dashboard

Each volunteer sees a personalized dashboard showing:

  • Upcoming Shifts - Quick view of scheduled commitments
  • Hours This Month - Track volunteer contributions
  • Zone Activity - Recent sightings in assigned zones
  • Quick Actions - One-click access to common tasks

Cancelled Shift Notifications: When shifts are cancelled, volunteers see them clearly marked with a red "Cancelled" badge and can dismiss them from their dashboard.

RippleVMS Feature Overview 5 / 14

Shift Browsing & Signup

Volunteers can easily find and sign up for shifts:

  • Filter by Zone - Find shifts in your coverage area
  • Filter by Type - Patrol, Collection, On-Call
  • List or Card View - Choose your preferred layout
  • One-Click Signup - Instant RSVP with confirmation
  • Calendar View - Week-at-a-glance scheduling

Automatic Notifications: Confirmation emails with calendar invites are sent automatically upon signup.

RippleVMS Feature Overview 7 / 14

Coordinator Tools

Efficient management for team leaders

Shift Creation & Management

Quick Shift Creation

Create shifts in under a minute with intuitive forms. Set zone, time, volunteer limits, and special requirements.

Roster Management

View who's signed up, confirm pending RSVPs, and manage volunteer assignments for each shift.

Bulk Actions

Select multiple shifts with "Select All" checkbox. Confirm pending RSVPs or cancel shifts in bulk.

Cancellation Workflow

Cancel shifts with automatic email notifications to all affected volunteers. Optional cancellation reason.

Shift+Click Selection: Hold Shift and click to select a range of shifts for bulk operations.

RippleVMS Feature Overview 11 / 14

Administration

Full system control and configuration

Admin Configuration

Zone Management

Create zones, assign counties, set colors, draw boundaries on the map, add Signal group links.

Shift Types

Configure shift types (Patrol, Collection, On-Call) with custom colors and default settings.

Qualified Roles

Define special roles like Verifier, Zone Lead, Dispatcher that volunteers can earn through training.

User Management

View all volunteers, update roles, assign zones, manage qualifications and training status.

Training Types

Set up training categories and requirements for volunteer certification.

System Settings

Configure organization name, feature flags, and system-wide preferences.

RippleVMS Feature Overview 13 / 14

Community Sighting Reports

Anonymous public reporting system for community safety:

  • No Login Required - Anyone can submit
  • S.A.L.U.T.E. Format - Standardized reporting
  • Location Picker - Map-based address selection
  • Media Upload - Attach photos if available
  • Instant Dispatch - Reports go to on-duty dispatchers

S.A.L.U.T.E.
Size - Activity - Location - Uniform - Time - Equipment
